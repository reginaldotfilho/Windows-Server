📄 💻 ETAPA 4 — CRIAR SCRIPT POWERSHELL
📍 4.1 Abrir bloco de notas
Windows + R → notepad
📍 4.2 Código do script
$url = "https://dl.google.com/chrome/install/GoogleChromeStandaloneEnterprise64.msi"
$destino = "C:\Temp\chrome.msi"

New-Item -ItemType Directory -Path "C:\Temp" -Force

Invoke-WebRequest -Uri $url -OutFile $destino

Start-Process "msiexec.exe" -ArgumentList "/i $destino /quiet /norestart" -Wait
📍 4.3 Salvar
Nome:
instalar_chrome.ps1
Tipo:
Todos os arquivos (*.*)
Local:
D:\Scripts

📄 📌 GPO: Bloquear uso de pendrive (USB)
🎯 Objetivo

Impedir que usuários utilizem dispositivos USB de armazenamento (pendrive, HD externo), aumentando a segurança dos dados da empresa.

🧠 📚 Conceito (pra sua apresentação)

Essa política bloqueia:

leitura de arquivos via USB
gravação de arquivos via USB

👉 Evita:

vazamento de dados
vírus/malware via pendrive
🚀 ⚙️ PASSO A PASSO COMPLETO
🧱 ETAPA 1 — Abrir o gerenciamento de GPO
Pressione:
Windows + R
Digite:
gpmc.msc
Pressione Enter
🧱 ETAPA 2 — Criar nova GPO
Vá em:
Objetos de Política de Grupo
Botão direito:
Novo
Nome:
GPO_Bloqueio_USB
✏️ ETAPA 3 — Editar a GPO
Botão direito na GPO
Clique:
Editar
📍 ETAPA 4 — Caminho completo (em português)
Configuração do Computador
> Políticas
> Modelos Administrativos
> Sistema
> Acesso a Armazenamento Removível
🔒 ETAPA 5 — Bloquear USB

Agora você vai configurar:

🔹 1. Bloquear leitura

Clique em:

Discos removíveis: negar acesso de leitura

👉 Marque:

Habilitado
🔹 2. Bloquear gravação

Clique em:

Discos removíveis: negar acesso de gravação

👉 Marque:

Habilitado
🔹 3. (Opcional, mais forte) Bloquear tudo
Todas as classes de armazenamento removível: negar todo acesso

👉 Habilitado

🔗 ETAPA 6 — Vincular a GPO
Vá na OU dos computadores
Botão direito:
Vincular um GPO existente
Escolha:
GPO_Bloqueio_USB
🔄 ETAPA 7 — Aplicar

No computador cliente:

gpupdate /force

👉 Reinicie o PC

⏳ RESULTADO
Usuário conecta pendrive
↓
Sistema bloqueia acesso
↓
Não consegue abrir nem copiar arquivos
✅ TESTE

✔ Conecte um pendrive
✔ Tente abrir

👉 Deve aparecer erro de acesso negado

⚠️ ERROS COMUNS

❌ Vincular na OU errada
❌ Não reiniciar
❌ Testar com usuário admin (às vezes ignora política)

🗣️ 💬 “Implementei política de segurança para bloqueio de dispositivos USB, impedindo leitura e gravação de dados em mídias removíveis, reduzindo riscos de vazamento e infecção por malware.”

📄 📌 GPO: Mensagem de aviso antes do login (banner corporativo)

Usando o Windows Server

🎯 Objetivo

Exibir uma mensagem obrigatória antes do usuário fazer login.

👉 Muito usado para:

aviso legal
política de uso
segurança da informação
🧠 💡 Conceito

Antes de digitar usuário e senha, aparece um aviso tipo:

“Este computador é de uso corporativo. Acesso monitorado.”

🧱 ETAPA 1 — Abrir GPO

gpmc.msc

GPO_Banner_Login

Você vai configurar DUAS opções:
